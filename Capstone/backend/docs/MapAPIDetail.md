# Schedule API 예시
## API 바로가기
| API 호출                             | 설명          |
|------------------------------------|-------------|
| [GET /map/category](#카테고리-조회)      | 카테고리 조회     |
| [GET /map](#지도-출력)                 | 지도 출력       |
| [GET /map/detail](#장소-세부-정보-출력)    | 장소 세부 정보 출력 |
| [GET /map/autocomplete](#장소명-자동완성) | 장소명 자동완성    |

### 카테고리 조회
